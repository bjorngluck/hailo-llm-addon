# Hailo LLM Add-on (v1.0.0)

Home Assistant add-on for **Hailo-10H + Model Zoo + Ollama**.

## v1.0.0 Release
- Better logging
- Built-in healthcheck
- Optional auto model download on first start

## Installation

1. Add repository:
   - Settings → Add-ons → Add-on Store → ⋮ → Repositories
   - Add: `https://github.com/bjorngluck/hailo-llm-addon`

2. Install **Hailo LLM** add-on

3. Configure options (optional):
   - `ollama_keep_alive`: How long to keep models loaded
   - `auto_download_model`: Download a default model on first start

## Requirements
- Raspberry Pi 5 + Hailo-10H AI HAT+
- Custom HAOS with Hailo-10H support
