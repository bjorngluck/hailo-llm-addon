# Hailo LLM Add-on

Home Assistant add-on for Hailo-10H + Model Zoo + Ollama.

## Installation

1. Add this repository:
   - Settings → Add-ons → Add-on Store → ⋮ → Repositories
   - Add: `https://github.com/bjorngluck/hailo-llm-addon`

2. Install **Hailo LLM**

## Requirements
- Raspberry Pi 5 + Hailo-10H
- Custom HAOS with Hailo support
